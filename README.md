# 飞书文档导出器

- 支持通过浏览器直接复制url下载文档及其子文档
  - 通过url自动判断是云文档还是知识库的文档
  - 支持指定导出文件的类型
  - 支持指定本地目录用于保存导出文件，自动创建目录
    - 云文档有目录，知识库没有（它的目录也是一个文档）
    - 导出云文档时，自动创建实际读取到的目录
      - 将不合法的目录字符串替换为下划线，如`/`替换为`_`
    - 导出知识库时，按文档层级自动创建目录，目录名为非叶子节点文档的标题
      - 自动创建的目录与源文档保存在同一级
      - 将不合法的目录字符串替换为下划线，如`/`替换为`_`
